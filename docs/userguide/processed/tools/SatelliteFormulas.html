Description of Formulas - Satellite

The following describes each formula included in the SEVIRI RGB section of McIDAS-V. These formulas are designed to work with SEVIRI MSG satellite data, although they are also applicable for equivalent channels of other imagers.

SEVIRI RGB

The SEVIRI RGB Manual band selection formulas allow you to select your desired wavelengths for each parameter. A recommendation for the wavelength to select for each field will be shown in the Field Selector window. The data from the wavelengths you select will be plugged into the formula for each parameter to produce the image in the Main Display window.

  • ACMP (cloud microphysics tuned for low cloud & fog) - Displays all-day (24 hour) cloud microphysics data tuned to mid-latitude low cloud and fog. This formula is based on absorbing IR channels only.
  • Field Definition

    IR8.7

    8.7 um IR Total Water, Cloud Phase, Dust

    IR10.8

    10.8 um IR Surface/Cloud-top Temp

    IR12.0

    12.0 um IR SFC/Cloud Temp, Low-level WV
  • AIRM (air mass) - Displays air mass data. This formula is based on absorbing IR channels only.
  • Field Definition

    WV6.2

    6.2 um IR Upper-level Water Vapor

    WV7.3

    7.3 um IR Mid-level Water Vapor

    IR9.7

    9.7 um Ozone

    IR10.8

    10.8 um IR Surface/Cloud-top Temp
  • DUST (cloud microphysics tuned for dust) - Displays cloud microphysics data tuned to mineral dust. This formula is based on absorbing IR channels only.
  • Field Definition

    IR8.7

    8.7 um IR Total Water, Cloud Phase, Dust

    IR10.8

    10.8 um IR Surface/Cloud-top Temp

    IR12.0

    12.0 um IR SFC/Cloud Temp, Low-level WV
  • GCMP (generic cloud microphysics) - Displays generic cloud microphysics data. After creating the display, you can change the numerical RGB range values. You can also set individual gamma values. This formula is based on absorbing IR channels only.
  • Field Definition

    IR8.7

    8.7 um IR Total Water, Cloud Phase, Dust

    IR10.8

    10.8 um IR Surface/Cloud-top Temp

    IR12.0

    12.0 um IR SFC/Cloud Temp, Low-level WV
  • GCON (generic enhancement of SCON) - Displays a generic enhancement of SCON. After creating the display, you can change the numerical RGB range values. You can also set individual gamma values. This formula is limited to use during the daytime.
  • Field Definition

    VIS0.6

    0.6 um VIS Cloud and Surface Features

    NIR1.6

    1.6 um Near IR Surface, Cloud Phase

    IR3.9

    3.9 um IR Low Cloud/Fog, Fire Detection

    IR6.2

    6.2 um IR Upper-level Water Vapor

    IR7.3

    7.3 um IR Mid-level Water Vapor

    IR10.8

    10.8 um IR SFC/Cloud-top Temp
  • NCMP (nighttime cloud microphysics, IR8.7 to IR3.9) - Displays nighttime cloud microphysics data.
  • Field Definition

    IR3.9

    3.9 um IR Low Cloud/Fog, Fire Detection

    IR10.8

    10.8 um IR Surface/Cloud-top Temp

    IR12.0

    12.0 um IR SFC/Cloud Temp, Low-level WV
  • NCOG (generic NCOL) - Displays a generic enhancement of NCOL. After creating the display, you can change the numerical RGB range values. You can also set individual gamma values. This formula is based on solar channels only.
  • Field Definition

    VIS0.6

    0.6 um VIS Cloud and Surface Features

    VIS0.8

    0.8 um VIS Aerosols over Water, Veg

    NIR1.6

    1.6um Near IR Surface, Cloud Phase
  • NCOL (natural colors) - Displays natural colors data. The minimum and maximum reflectivity values are used to rescale the color beams. The minrefl and maxrefl values should be between 0 and 100. This formula is based on solar channels only.
  • Field Definition

    minrefl

    Minimum Reflectivity Limit (% reflectivity)

    maxrefl

    Maximum Reflectivity Limit (% reflectivity)

    VIS0.6

    0.6 um VIS Cloud and Surface Features

    VIS0.8

    0.8 um VIS Aerosols over Water, Veg

    NIR1.6

    1.6 um Near IR- Surface, cloud phase
  • SCON (severe convection, daytime only) - Displays severe convection data. This formula is limited to use during the daytime.
  • Field Definition
